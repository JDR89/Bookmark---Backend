## Teslo API

1.Clonar proyecto

2.Instalar dependencias
__npm install__

3.Crear archivo .env a partir de template
__.env.template .env__

4.Configurar archivo .env con tus credenciales de base de datos

5.Levantar la base de datos
__docker-compose up -d__

6.Iniciar la aplicación
__npm run start:dev__

7.Ejecutar seed
__http://localhost:3000/api/seed__
